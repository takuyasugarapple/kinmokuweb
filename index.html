代表声明

自分は、1人でプログラミングの学習を継続して行うことができず、挫折してしまいました。
幸運なことに作りたい物ができ、４年時にもう一度学習を始めました。
そんな時に自分と同じ思いを抱えている人がいるはずだと思い、「近もく会」を創りました。
やりたいと思った時にいつでも参加でき、チャレンジする人を暖かく迎え入れる場所であって欲しいなあ。

近もく会代表 中原雄太

TEAM近もく

近もく会は様々なバックグラウンドを持つ人々が自主的に集って学習し、プログラミングという能力を用いて自ら周りの人に“何か“を与えようとする人々の集まりです。

チームメンバーのバックグラウンドも様々で、ビジネスを始めるためにプログラミングをやっている人、デザインを学びながら学習している人、好奇心で動いて色々吸収したいから学んでいる人など。モチベーションも様々ですが目的は同じ人たちです。

活動

各自の勉強をしたり、教えあったりが主な活動内容です。
また、外部からエンジニアの方をメンターとしてお招きして、教えいただいております。
大学内でのイベントの企画もこれから行っていくつもりです。

近もく会学習言語：HTML / CSS / JavaScript / jQuery / Swift / Ruby / Ruby on Rails / PHP / Laravel / Python / Java / Kotlin / Rust

主に勉強している言語

HTML&CSS

HTMLは、ページの要素や構造を指定するための言語です。
CSSはHTMLで作った要素を装飾するスタイルを指定する言語です。

Design

近もく会ではappデザイン、webデザイン、フライヤーデザインなどをメインとして勉強・活動しています。

Swift

apple社が開発したiphoneアプリを開発するには不可欠な言語です。近もく会代表の中原もこの言語を勉強しており、iOSアプリを作りたいならこの言語がオススメです。

PHP

HTMLを動的に表示させること。サーバサイドでデータを扱った処理などができます。websiteを作りたいならこの言語がオススメです。
